# homebrew-tap

Homebrew tap for [ai-notify](https://github.com/unoryota/ai-notify) — notifications for terminal AI coding agents.

```sh
brew install unoryota/tap/ai-notify
```

Then:

```sh
ai-notify init        # wire your agents (Claude Code / Codex / …)
ai-notify menubar install   # native menu bar (macOS)
```

See the [ai-notify README](https://github.com/unoryota/ai-notify) for full docs.
